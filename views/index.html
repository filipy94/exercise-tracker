Exercise Tracker

Build a full stack JavaScript app that is functionally similar to this: FCC URL Exercise Tracker Model.

Create a New User

POST /api/users

Add Exercises

POST /api/users/:_id/exercises

GET user's exercise log: GET /api/users/:_id/logs?from=value&to=value&limit=value

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
